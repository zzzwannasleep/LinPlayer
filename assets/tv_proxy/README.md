# TV Proxy Assets

This folder is used to bundle **mihomo** (Android TV) and **metacubexd** (web UI) into the APK.

Expected files (generated by `tool/fetch_tv_proxy_assets.ps1`):

- `assets/tv_proxy/mihomo/android/arm64-v8a/mihomo.gz`
- `assets/tv_proxy/mihomo/android/armeabi-v7a/mihomo.gz`
- `assets/tv_proxy/mihomo/android/x86_64/mihomo.gz` (optional)
- `assets/tv_proxy/mihomo/android/x86/mihomo.gz` (optional)
- `assets/tv_proxy/metacubexd/compressed-dist.tgz`

Notes:
- CI builds run the fetch script automatically.
- If you hit GitHub API rate limits locally, set `GITHUB_TOKEN` or `GH_TOKEN` before running the script.
